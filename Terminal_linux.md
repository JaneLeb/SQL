# УПРАВЛЕНИЕ БД ПРИ ПОМОЩИ ТЕРМИНАЛА

## Шаг 1. Откройте терминал и подключитесь к PostgreSQL

Введите следующую команду и нажмите Enter:
Если Вы работаете на Linux, запускайте подключение через команду `sudo`. Это переключит Вас на системного пользователя `postgres`, у которого есть прямой доступ:

bash

```
sudo -u postgres psql -d pizza_db
```
Пароль: Терминал запросит пароль — введите пароль от вашего компьютера (от учетной записи Linux jane) и нажмите Enter.Флаг -d pizza_db сразу подключит вас к нужной базе данных, которую видно на вашем скриншоте.

-   *Если при установке вы создали другую базу данных или пользователя, замените `postgres` на ваше имя.*
-   *Если утилита запросит пароль, введите пароль, который вы указывали вчера при установке PostgreSQL (при вводе символы отображаться не будут, это нормально) и нажмите Enter.*

Когда подключение выполнится успешно, строка ввода изменится и станет похожа на `postgres=#`.

Теперь вы находитесь внутри интерактивной консоли PostgreSQL. Строка ввода должна выглядеть как `pizza_db=#`.



## Шаг 2. Как выполнить запрос или файл

Вот как создавать и выполнять запросы прямо в терминале:


✍️ Как писать запросы вручную

Вы можете писать SQL-команды прямо в консоли.

1.  Введите ваш запрос (например, `SELECT * FROM имя_таблицы`). Здесь нажатие Enter просто переносит строку.
2.  В конце запроса обязательно поставьте точку с запятой `;` и только после этого нажмите Enter. Без `;` робот внутри терминала будет думать, что вы еще продолжаете писать команду.

-   Ввод команды: Просто набирайте SQL-код, например: `SELECT name, age FROM person WHERE address = 'Kazan';`
-   Важное правило: Запрос выполнится только после того, как вы поставите точку с запятой `;` и нажмете Enter.
-   Перенос строк: Если вы нажмете Enter без `;`, терминал не выполнит код, а просто перенесет курсор на следующую строку (в начале строки появится знак `->`). Это сделано для того, чтобы можно было красиво оформлять длинные запросы.



📁 Как запустить готовый файл `.sql` (например, day00_ex00.sql)
 (самый удобный способ)\
Если вы хотите выполнить весь код из файла `day00_ex00.sql`, внутри консоли `psql` введите команду `\i` и укажите полный путь к файлу:

sql

```
\i /путь_к_папке/day00_ex00.sql

```

-   *Подсказка: Чтобы не писать путь вручную, напишите `\i` , а затем просто перетащите файл мышкой из папки прямо в окно терминала.*

Вместо ручного копирования кода используйте команду мета-команду `\i`.

1.  Наберите: `\i`
2.  Перетащите файл `day00_ex00.sql` мышкой из папки прямо в окно терминала (путь подставится автоматически).
3.  Нажмите Enter. Терминал сам прочитает файл и сразу выдаст результат на экран.

🧭 Полезные команды для навигации

-   `\dt` --- показать список всех таблиц в базе данных (убедитесь, что там есть `person`).
-   `\d имя_таблицы` --- посмотреть структуру конкретной таблицы (какие в ней есть колонки и типы данных).
-   `\q` --- выйти из консоли базы данных обратно в обычный терминал Linux.




## Шаг 3. Как выйти из терминала

Чтобы закрыть сессию PostgreSQL и вернуться в обычную консоль, введите:

sql

```
\q

```

Используйте код с осторожностью.
